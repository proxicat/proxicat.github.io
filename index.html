Frequently asked questions

How long does setup take?

Less than 5 minutes. Create a profile, enter your proxy address, toggle which dev tools you want, and click the power button. That's it.

Is it really $19 one-time? No subscription?

Yes. One purchase, use forever. No monthly fees. Free updates within the major version. 14-day free trial to try before you buy.

What if it breaks my network settings?

ProxiCat automatically backs up your system settings on first launch. One-click restore returns everything — env vars, system proxy, DNS, git config, npm config, /etc/hosts — to exactly how it was before.

Can I get a refund?

Yes. 30-day money-back guarantee, no questions asked.

Does it work with my proxy client (Clash, V2Ray, Shadowsocks, etc.)?

Yes. ProxiCat works with any local proxy that listens on HTTP or SOCKS5. It auto-detects 30+ known proxy clients and validates that ports match your configuration.

Why not just use System Preferences?

System proxy only covers apps that read macOS proxy settings. Many CLI tools (git, npm, curl) and some GUI apps ignore it. ProxiCat covers 4 layers: shell env, system proxy, per-tool config, and system-level interception.

Does it support Apple Silicon (M1/M2/M3/M4)?

Yes. ProxiCat is built natively with SwiftUI for arm64. No Rosetta needed. Runs on macOS 15 (Sequoia) and later.
